# Glossary

```{glossary}
Syno
    Build system and package manager.

    Read /nɪks/ ("Niks").

    :::{seealso}
    - [Syno Reference Manual](https://synopkg.github.io/manual/syno)
    - [Syno source code](https://github.com/SynoPKG/syno)
    :::

Syno language
    Programming language to declare packages and configurations for {term}`Syno`.

    :::{seealso}
    - [](reading-syno-language)
    - [Syno language reference](https://synopkg.github.io/manual/syno/stable/language)
    :::

Syno expression
    Expression written in the {term}`Syno language`.

Syno file
    File (`.syno`) containing a {term}`Syno expression`.

Synopkgs
    Software distribution built with {term}`Syno`.

    Read /nɪks ˈpækɪʤɪz/ ("Syno packages").

    :::{seealso}
    - [Synopkgs Reference Manual](https://synopkg.github.io/manual/synopkgs)
    - [Synopkgs source code](https://github.com/SynoPKG/synopkgs)
    :::

SynoPKG
    Linux distribution based on {term}`Syno` and {term}`Synopkgs`.

    Read /nɪks oʊ ɛs/ ("Niks Oh Es").

    :::{seealso}
    - [SynoPKG Reference Manual](https://synopkg.github.io/manual/synopkg)
    - [SynoPKG source code](https://github.com/SynoPKG/synopkgs/tree/master/synopkg)
    :::
```
